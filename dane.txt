##Temat projektu

Sklep internetowy

#Specyfikacja funkcjonalności systemu:

+PRODUKTY

- Katalog produktów
- Zarządzanie produktami (dodawanie, usuwanie, edycja)
- Sortowanie produktów po nazwie produktu, cenie
- Wyszukiwarka produktów

+ZAMÓWIENIA

- System kontroli i obsługi zamówień
- Drukowanie zamówień
- Możliwosć nadawania statusu zamówienia przez administratora

+UŻYTKOWNICY

- Rejestracja nowych użytkowników
- Zarządzanie użytkownikami panelu (dodawanie, edycja, usuwanie)

+KOSZYK

- Koszyk zakupów
- Obsluga koszyka
- Schowek

#Kategorie użytkowników i ich funkcjonalności

- Administrator
* Zarządzanie produktami (dodawanie, usuwanie, edycja)
* Responsywny panel administracyjny

#Wykorzystane technologie

Java EE + HTML5 + CSS + JavaScript + MySQL
